# Advanced Dynamical Systems Portfolio

Portfolio of applied machine learning systems built around **state space modeling, Kalman filtering, hybrid ML, and causal inference**.

These projects demonstrate how latent-state modeling can be applied to real-world dynamic systems across multiple industries.

---

# Core Modeling Framework

All projects follow a common architecture:

1. Latent State Modeling (State Space Models)
2. Kalman Filtering for state estimation
3. Feature Engineering from filtered signals
4. Hybrid ML Risk Prediction
5. Causal Intervention Analysis
